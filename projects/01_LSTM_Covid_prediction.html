📁 졸업논문: [LSTM 코로나 확진자수 예측 프로그램]

📝 프로젝트 개요

딥러닝 모델인 LSTM(Long Short-Term Memory)을 활용하여 코로나 확진자수를 예측하고, 모델 성능을 개선하기 위해 외부 변수(사회적 거리두기, 평균기온) 및 해외 확진자 데이터를 결합한 학사 졸업 프로젝트입니다.

⏱️ 기간 및 인원

기간: 2023.09 ~ 2023.12

인원: 개인 프로젝트

🛠 기술 스택

🔧 핵심 기능

  • ✔ 사회적 거리두기 정도, 평균 기온을 변수로 추가하여 예측 성능 향상
  • ✔ 일본·러시아 확진자 데이터를 추가한 LSTM 모델 학습
  • ✔ 시계열 예측을 위한 단일/다중 입력 데이터 구조 구현
  • ✔ MAE 기반 성능 평가 및 오차 분석

🐛 트러블 슈팅

  • ✔ 해외 데이터 병합 시 성능 저하 → 각국의 전염병 패턴 차이로 인한 학습 어려움 확인
  • ✔ 평균 기온 변수를 추가했을 때 오히려 모델 정확도 저하 → 영향력 낮은 변수는 오히려 방해 가능성

🌟 팀 성과

  • ✔ 사회적 거리두기 변수만 추가했을 때 MAE 242로 가장 높은 성능 확인
  • ✔ "내일 확진자 추세는 오늘과 유사"라는 전제에 대한 검증 실험 포함

🧑‍💻 개인 기여

  • ✔ 전체 실험 설계 및 데이터 수집/전처리
  • ✔ LSTM 모델 구조 정의 및 학습
  • ✔ 변수 실험 및 결과 해석 보고서 작성

💡 깨달은 점

데이터를 단순히 늘리는 것보다, 실제 문제에 영향을 주는 요인을 정확히 분석하는 것이 중요하다는 것을 체감했습니다. 예측 모델 성능 향상을 위해서는 변수의 질적 선택과 정교한 전처리 설계가 필수적이라는 점을 배웠습니다.
