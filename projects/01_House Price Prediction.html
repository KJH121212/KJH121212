🏠 아파트 실거래가 예측 프로젝트

📝 개요

본 프로젝트는 서울시 아파트 실거래가 데이터를 기반으로 아파트 가격을 예측하는 머신러닝 모델을 개발한 팀 프로젝트입니다. 다양한 외부 데이터를 활용한 파생변수 생성 및 모델 최적화를 통해 RMSE 성능을 개선하고 대회 최종 2위를 기록하였습니다.

🧪 실험 개요

  • 모델: XGBoost 기반 회귀 예측 모델
  • 환경: Jupyter Notebook, VSCode, Ubuntu
  • 데이터: train/test dataset (40여 개 feature 포함), 버스/지하철 위치 데이터
  • 기법: 로그 변환, 결측치 처리, 레이블 인코딩, 거리 기반 파생변수 등

📊 실험 구성

  • 동별 평균가격 파생변수 생성으로 지역 기반 가중치 부여
  • 지하철역, 버스정류장과의 거리 기반 파생변수 활용
  • 하이퍼파라미터 튜닝: RandomizedSearchCV + EarlyStopping

🧠 결과 요약

  • RMSE 기준 최종 2위 성과 달성 (11628.7728)
  • 하이퍼파라미터 조정 및 Feature Engineering으로 성능 향상
  • 파생변수의 영향도 분석을 통해 모델 해석력 제고
