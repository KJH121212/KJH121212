📊 졸업논문: LSTM 기반 코로나 확진자 수 예측

📝 개요

본 논문은 딥러닝 기반의 LSTM(Long Short-Term Memory) 모델을 사용하여 대한민국의 코로나19 확진자 수를 예측하고, 그 성능을 향상시키기 위한 두 가지 전략(외부 변수 추가, 해외 확진자 데이터 포함)을 실험적으로 검토한 학사 졸업 논문입니다.

🧪 실험 개요

  • 모델: LSTM 기반 시계열 예측 모델
  • 데이터 출처: Johns Hopkins, 기상청, KRIHS
  • 입력 변수: 확진자 수, 사회적 거리두기 단계, 평균 기온
  • 학습 방식: One-step 예측 / Multi-step 예측은 미사용

📊 실험 구성

  • 실험 A-C: 한국 확진자 + 외부 변수
  • 실험 D-G: 해외 확진자(일본/러시아) 추가 → 학습 데이터 확장

🧠 결과 요약

  • 사회적 거리두기 변수를 추가하면 예측 정확도(MAE, Loss)가 향상됨
  • 평균 기온 변수는 큰 영향 없음 → 학습 성능 오히려 저하
  • 해외 데이터 추가 시 러시아 데이터는 오히려 예측 성능 악화
  • 결론: 외부 변수 추가는 효과적, 해외 데이터 확장은 부정적 영향 가능

📁 논문 정보
